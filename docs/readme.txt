一、对接接口说明
1、注册（用户）
post地址http://192.168.0.101:5830/open/getLoginChat4Url
参数{productKey:'123',userKey:'101',userName:'zhang01',age:'25',sex:'MALE',loginName:'zhang01',password:'123456'}
productKey：产品key，DBA提供
userKey：用户的唯一标识
其他参数都是附加参数
返回结果{code:0,msg:'success',user_url_param:'dfd51f705f6911e68d38371176b4e664'}
2、登录（用户）
get地址192.168.0.101:5830/user/otherLogin?id=7de551f0645411e688d27bafa03ed49e
7de551f0645411e688d27bafa03ed49e是user_url_param
时效性30分钟
3、注册（管理员）
post地址http://192.168.0.101:5830/open/getAdminLoginChat4Url
参数{productKey:'123',adminKey:'201',admin_name:'admin01',loginName:'admin01',password:'123456'}
productKey：产品key，DBA提供
adminKey：管理员的唯一标识
其他参数都是附加参数
返回结果{code:0,msg:'success',admin_url_param:'dfd51f705f6911e68d38371176b4e664'}
4、登录（管理员）
get地址192.168.0.101:5830/coumg/otherLogin?id=7de551f0645411e688d27bafa03ed49e
7de551f0645411e688d27bafa03ed49e是admin_url_param
时效性30分钟
5、咨询师登录地址
192.168.0.101:5830/counselor/
二、退出页面配置
configFiles/config
/*退出页面配置*/
exports.loginOutConfig={
    user:'http://www.baidu.com',
    coumg:"http://www.163.com"
}
不配置就写空值，不能不写，用户user，管理员coumg，如：
/*退出页面配置*/
exports.loginOutConfig={
    user:'',
    coumg:""
}
三、定时任务的配置文件
/controllers/schedule
四、sql语句全都是在/daoSQL/*中的xml文件中写着
/configFiles/sqlMapConfig文件中配置xml文件所在位置
注意：xml文件中<%%>和{%%}的关系
xml中的sql语句中动态数据的语法是ejs的语法规则，并把<%%>换成了{%%}，最后这些语句会用ejs渲染成真正的sql语句
